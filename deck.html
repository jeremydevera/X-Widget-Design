00assumptions / open questions
name
xcode — note: this is Apple's IDE trademark since 2003. fine for prototyping, must change for App Store. swap when ready.
audience
power users / iOS devs / streamers. comfortable with system.cpu namespacing & tabular numerics.
island slots
L1 L2 R1 R2 — 4 mappable positions. compact = 1 metric, split = 2 (one per side).
data shape
all metrics expose {value, unit, threshold, trend}. sparkline = last 60s @ 1Hz sample.
aesthetic guard
monochrome only. no hue at all. state encoded by brightness + glyphs (▲) + inverted-pill (white-on-black → black-on-white for crit). no glassmorphism, no emoji.
scope cut
live activity is the only surface in v1. lock screen widgets / home widgets = v2.
